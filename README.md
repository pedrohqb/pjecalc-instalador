# pjecalc-instalador

Script que descompacta o .exe do Pjecalc e faz funcionar no Linux sem precisar do Wine.

Instale o pacote pjecalc-instalador, faça o download do Pje Office em formato .exe, abra no menu do sistema o instalador e selecione o arquivo .exe. A instalação é rápida; logo após basta abrir no menu o Pjecalc.

Depende de 7zip, kdialog, zulu-8, fontconfig e ttf-mscorefonts-installer. O Zulu pode ser baixado em https://www.azul.com/downloads/?version=java-8-lts&package=jdk#zulu. Caso utilize Debian, é necessário habilitar o repositório contrib.
